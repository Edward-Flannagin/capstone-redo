# Capstone Redo

A modern React web application built for capstone project requirements. This application features a responsive Bootstrap-based UI with interactive components for an enhanced user experience.

## 🚀 Features

- **React 18** - Latest React framework for building dynamic user interfaces
- **Bootstrap 5** - Responsive UI components and styling
- **React Router** - Client-side routing for seamless navigation
- **Date Picker** - Interactive date selection with `react-datepicker`
- **Notifications** - Toast notifications using `react-toastify`
- **Animations** - Celebration effects with `react-confetti`
- **Testing** - Comprehensive test suite with React Testing Library
- **Responsive Design** - Works beautifully on desktop, tablet, and mobile devices

## 📋 Prerequisites

Before you begin, ensure you have the following installed:
- **Node.js** (v14 or higher)
- **npm** (v6 or higher) or **yarn**

## 🛠️ Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Edward-Flannagin/capstone-redo.git
   cd capstone-redo
   ```

2. **Install dependencies**
   ```bash
   cd capstone-redo
   npm install
   ```

## ▶️ Running the Application

### Development Mode
Start the development server with hot reload:
```bash
npm start
```
The application will open in your browser at [http://localhost:3000](http://localhost:3000)

### Production Build
Create an optimized production build:
```bash
npm run build
```
The build folder will contain the production-ready files.

### Running Tests
Launch the test suite in interactive watch mode:
```bash
npm test
```
